# react-native-skia-ui

Cross platform UI primitives implemented with `@shopify/react-native-skia`

## Installation

```sh
npm install react-native-skia-ui
```

## Usage

```js
import { multiply } from 'react-native-skia-ui';

// ...

const result = await multiply(3, 7);
```
